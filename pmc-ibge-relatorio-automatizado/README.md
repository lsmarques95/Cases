# PMC IBGE - Relatório Automatizado

## Sobre o projeto
Projeto desenvolvido para automatizar a geração de um relatório da Pesquisa Mensal do Comércio (PMC) do IBGE.

## Objetivo
Automatizar a coleta, tratamento e apresentação dos dados da PMC, gerando tabelas e gráficos de forma replicável.

## O que o projeto faz
- Coleta dados da PMC
- Trata e organiza as séries
- Gera tabelas com indicadores
- Cria gráficos setoriais
- Produz relatório final em PDF

## Fonte dos dados
IBGE / SIDRA

## Tecnologias utilizadas
- Python
- pandas
- requests
- xlwings

## Resultado
Geração automática de relatório mensal com tabelas e gráficos da PMC.
