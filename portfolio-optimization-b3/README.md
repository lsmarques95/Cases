# Portfolio Optimization - B3

## Sobre o projeto
Projeto desenvolvido para construção e avaliação de uma carteira de ações brasileiras utilizando dados históricos da B3.

O trabalho envolve a criação de uma carteira equally weighted, cálculo de métricas de risco e retorno, comparação com o IBOVESPA e análise quantitativa de performance.

## Objetivo
Avaliar o desempenho de uma carteira com pesos iguais e propor melhorias na alocação com base em métricas quantitativas.

## Metodologia
- Construção de carteira equally weighted
- Cálculo de retorno acumulado
- Cálculo de retorno anualizado
- Cálculo de volatilidade anualizada
- Cálculo do índice de Sharpe
- Cálculo do máximo drawdown
- Cálculo do beta em relação ao IBOV
- Comparação com benchmark
- Backtest da estratégia

## Métricas avaliadas
- Retorno anualizado
- Volatilidade anualizada
- Sharpe ratio
- Maximum drawdown
- Beta vs IBOV

## Tecnologias utilizadas
- Python
- pandas
- numpy
- matplotlib
- statsmodels
- Jupyter Notebook

## Resultados
O projeto gera:
- curva de retorno acumulado da carteira
- comparação com IBOVESPA
- métricas de risco e retorno
- avaliação quantitativa da estratégia

## Fonte dos dados
Dados históricos de ações da B3, IBOVESPA e CDI.
